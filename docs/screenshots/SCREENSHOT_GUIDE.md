# Chrome Web Store Screenshot Guide

Use this exact flow to capture listing screenshots quickly.

## Required size

Use one of these dimensions for each screenshot:
- 1280 x 800 (recommended)
- 640 x 400

## Capture setup

1. Open Gmail in Chrome (`https://mail.google.com`).
2. Open DevTools device toolbar if needed and set a fixed viewport to 1280 x 800.
3. Ensure inbox data in screenshot is safe to share (or blur sensitive data).
4. Turn on your extension and open the Gmail unsubscriber sidebar.

## Shot list (recommended)

1. `01-sidebar-overview.png`
   - Gmail inbox with Gmail unsubscriber sidebar visible.
2. `02-select-like-open-email.png`
   - Open email + button `Select All Emails Like Open Email` highlighted.
3. `03-archive-actions.png`
   - Show archive action buttons in sidebar.
4. `04-safety-message.png`
   - Show a meaningful safety/log message in the sidebar.
5. `05-complete-state.png`
   - Show success state/toast after action.
